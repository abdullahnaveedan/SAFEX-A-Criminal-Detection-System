SAFEX- A Criminal Detection System

The Criminal Detection System in Machine Learning is an emerging technology that has the potential to revolutionize law enforcement in Pakistan. With a high crime rate and limited resources, the use of machine learning in criminal detection could help police departments to more effectively and efficiently combat crime. It utilizes data from criminal records, social media, and surveillance footage to analyze patterns and identify potential criminal activity. It employs advanced algorithms and artificial intelligence to learn and adapt to changing criminal image, allowing law enforcement to stay ahead of the curve. To recognize faces in images, the Criminal Detection System in Machine Learning uses a well-known Face Detection Technique called MTCNN. The Convolutional Neural Network (CNN) is used to identify the person using the Face-Net model, frequently used in face recognition tasks, if the MTCNN algorithm successfully detects a face in a particular age region of database.

Problem

Currently the system is only available for high crime areas and not available in Pakistan. Only UK china and UAE use this system but in real time environment likes Riots only drone mobile camera CCTV can work.

Expected Results

An individual can be accurately identified from a crowd through the implementation of AI and machine learning techniques. This advancement aids law enforcement agencies in efficiently locating suspects.

Outcomes

This software facilitates the detection of criminals using cameras and CCTV systems. Its potential benefits extend to safeguarding educational institutions, corporate entities, and public gatherings ( social and political )
